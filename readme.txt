qc2texについて

説明
 qc2texは、アセンブリ風の量子回路の記述から、量子回路図、量子回路の振る舞いのシミュレーションを行うものです。
 in.txtに記述された内容から、out.texというtexファイルを作成します。
 独自実装の文字式演算アルゴリズムを搭載しています。
  基本的に授業の自習用に作られたもので、対象となる量子回路が正常に動作することを主眼として制作されています。そのためいくつか問題のある実装も目立ちます。

利用方法
 in.txtを、初期状態のin.txtを参考に作成してください。同じファイルは/in.txt.sample/sample.txtに存在します。
 /in.txt.sample/内には他のサンプルコードもいくつか存在します。使用する場合各ファイルをin.txtとリネームしてqc.plと同じフォルダ内においてください。
 qc.plを実行すると、out.texが作成されます。
  参考:WindowsではPerlはファイルを開く要領で実行出来ます。Macではシェル上で"perl (対象ファイル名)"で実行出来ます。Lunuxでは同じ方法か、シェルから直接指定すれば動作する場合が多いですが、ファイル属性を実行可能にする必要がある場合もあります。"chmod 755 (対象ファイル名)"で実行可能になります。
 また、Windows環境では、test.batを実行する事でpdfを作成し結果を確認出来ます。Unix環境(Mac,Linux)ではtest.shが動くかもしれません。(テストはしていません。)

既知の問題
 C?GATE(量子制御ユニタリゲート)が正常動作しません。
 虚数の扱いで問題が生じる事があります。
 文字式計算アルゴリズムのテストが不十分で予想外の動作をする可能性があります。
 エラーおよび古典ビット演算を量子回路図で適切に表示出来ません。
 量子回路図が自動的にリサイズされません。
 ビット数を増やす毎に計算量が爆発的に増大します。(量子ビットでは当然ですが、古典ビットの場合も同様になる事は問題です。)
 エンタングルメントを考慮した表示に対応していません。
  これらへの対応は一部は簡単でそれほど時間がかかりません。必要であれば対処できます。

更新履歴
 0.10 初期バージョンです。既知の各種の問題に対処後1.0になる予定です。

必須環境
 Active Perl等Perl5の実行環境。
 Platex等texコンパイル環境。
  参考:東京大学のMac環境では標準で問題なく動作します。多くのLinux環境でも問題ないと思われます。Windowsの場合は標準ではどちらも導入されていないので独自にインストールする必要があります。

ライセンス
 再配布、改造、実行、組み込み、等基本的に自由です。完全オリジナルではない事のみ明示を願います。
 計算の誤り、ファイルの破損など問題発生時は一切責任を持ちません。
 文字式計算アルゴリズムmath.plは同作者の作成で同じライセンスが適用されます。
 Jcode.pmは歌代和正によるもので、そちらのライセンスが適用されます。本作者の作成ではありません。このことの明示の元再配布が許可されています。

著作権
 再配布可能のため、本名の記載は行いません。メールによる送信者に著作権等が存在します(Jcode.pm除く)。
 必要な場合は'kurema'の名前を用いて、作者の明示をお願いします。
